# Omnichunk (Updated for Minecraft 1.21)

A Minecraft mod that causes changes to one chunk to affect ALL chunks!

## Building

1. Install JDK 21
2. Run `./gradlew build`
3. Find JAR in `build/libs/`

## Requirements

- NeoForge 1.21.3+
- Minecraft 1.21

## Notes

Ported from 1.16.5 to 1.21.
